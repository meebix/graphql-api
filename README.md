# Node API

Built with Express.js, REST & GraphQL, TypeORM, and MySQL.

**Features:**

- Auth'z & Auth'n (RBAC / JWT)
- Security (CORS, CSRF, CSP)
- Flexible Config
- Thorough Error Handling
- Request Logging (Bunyan)
- Request Timer
- Health Check
- Mailer using Sparkpost
- SMS using Twilio
- Testing (Unit, Integration)
- Linting
